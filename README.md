# E-Canteen
**E-Canteen is an online food ordering and payment application at the Pamulang 2 University canteen**<br />

## What Have I Done
- Slicing design to code using Jetpack Compose
- State Hoisting
- MVVM Architecture
- Shared Preference
- Restful API
- Building API with PHP native
- MySQL
